# Trang bán giày
Một ứng dụng được xây dựng bằng React JS để thực hành tư duy Component-based và xử lý dữ liệu từ API.

### ⚛️ Công nghệ và Thư viện:
* **Core:** React JS (Functional Components)
* **Routing:** React Router DOM (để chuyển trang)
* **API Handling:** Axios / Fetch API
* **Styling:** CSS Modules / Tailwind CSS

### 🌟 Tính năng nổi bật:
* **Component Reusability:** Chia nhỏ giao diện thành các thành phần (Header, Card, Button) để dễ dàng tái sử dụng.
* **Responsive Design:** Giao diện hiển thị tốt trên mọi thiết bị từ Mobile đến Desktop.

### 🛠 Cách chạy dự án:
1. Mở link CodeSandbox: [https://8vph3q-3000.csb.app/]
2. Hoặc tải về máy và chạy:
   - `npm install`
   - `npm start`
